### How to run the demo
---

#### step 1
Create and Enter the folder `netflixDemo`:   

	mkdir netflixDemo && cd ./netflixDemo
  
#### Step 2
clone the repository: 

	git clone https://github.com/ChenyuSu624/NetflixDemo.git


#### Step 3
run it: 
	
	yarn install && yarn start

#### Step 4
Open the Browser:
	
	http://localhost:3000
